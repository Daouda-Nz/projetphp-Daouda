# Projet POO PHP – Gestion des Utilisateurs

## 📌 Objectif

Ce projet a pour objectif de mettre en pratique les concepts de la programmation orientée objet en PHP.

## 🧠 Concepts utilisés

* Classes et objets
* Encapsulation
* Héritage
* Classes abstraites
* Interfaces
* Polymorphisme
* Attributs statiques
* Constantes

## 👥 Types d'utilisateurs

* Client
* Employé
* Administrateur

## ⚙️ Fonctionnalités

* Connexion des utilisateurs
* Calcul de réduction (Client)
* Calcul de salaire annuel (Employé)
* Suppression d'utilisateur (Administrateur)
* Affichage des informations
* Comptage du nombre total d'utilisateurs

## 🗂️ Structure du projet

* classes/

  * Personne.php
  * Utilisateur.php
  * Client.php
  * Employe.php
  * Administrateur.php
  * Interfaces.php
* index.php

## ▶️ Exécution

Lancer le projet via un serveur local (WAMP/XAMPP) :

http://localhost/Projetphp-Daouda/index.php
